1 创建分支
	git  branch   branchname
2 切换分支
	git  checkout   branchname:
3 删除分支
	git   branch   -d   branchname  
4 强制删除  与-d的区别是 如果部分更改没提交 直接强制删除
	git branch -D devlop2  #  delete branch   
5 合并分支  首先切换到要合并的分支上面
	git  merge    branchfilename
